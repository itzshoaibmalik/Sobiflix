About Sobiflix

Our Mission

Sobiflix aims to be your go-to destination for discovering and streaming a diverse range of movies, captivating dramas, and exciting anime. We want to create a seamless and visually appealing platform for entertainment lovers.

What We Offer

  • A curated selection of Movies, Anime, and Dramas powered by the TMDb API.
  • High-definition streaming integration capability (Note: Actual streaming not implemented in this example).
  • Constantly updated content like Trending, Now Playing, and Top Rated titles.
  • A user-friendly interface designed for easy browsing and discovery.

About the Creator

Hi! I'm Shoaib Malik, the creator behind this Sobiflix concept.

I'm currently a first-year undergraduate student pursuing my passion for technology at Chitkara University. This project is an exploration of front-end development, UI/UX design, and working with external APIs like TMDb.

I'm constantly learning and exploring new technologies to build engaging and functional web experiences. Feel free to connect!

Technology Stack (Concept)

This demonstration utilizes:

  • HTML5 for structure.
  • CSS3 for styling, including modern layouts like Flexbox and Grid.
  • Vanilla JavaScript for DOM manipulation and API interactions.
  • The Movie Database (TMDb) API for fetching movie and TV show data.
  • Font Awesome for icons.

Note: This is a front-end concept and does not include backend authentication, database storage, or actual video streaming infrastructure.
